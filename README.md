# homework-planner

An interactive single-file HTML application for organizing university coursework over **13 weeks**.

## Features
- 📅 **Weekly tasks** for:
  - Linear Algebra 1 (homework + 5 random graded bonus tasks)
  - Analysis 1 (homework, quiz, bonus task each week)
  - Chemistry (graded bonus tasks + exercises)
  - Mechanics (homework)
  - Computer Science (homework)
  - Engineering Design (homework)
- 🔁 **Weekly recap task** per subject
- ✅ Track **due date, done date, and completion status** for every task
- ✏️ Add personal **notes** per row
- 🔎 **Filter by week or subject**
- ⬇️ **Export to CSV** (saves visible rows)
- 🧹 Quick action: clear all “Done” checkboxes

## Usage
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start filling in your tasks, deadlines, and progress.
